# ospinajuanp-single-page

This is an unofficial template for creating single-page React applications. It offers a modern build setup without a lot of configuration.

## Install

```npm
npm install ospinajuanp-single-page
```
```npm
ospinajuanp-single-page
```

# Usage
## Development Serve
```bash
npm run dev
```
## Development
```bash
npm run start
```
## Production
```bash
npm run build
```
